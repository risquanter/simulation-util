# Contributing to Risquanter - Simulation-Util

This project uses DCO to manage contributions:

## Licensing and Contributions

### Developer Certificate of Origin (DCO v1.1)
All contributions must be signed-off, certifying the Linux Foundation's Developer Certificate of Origin version 1.1
(https://developercertificate.org/). This certifies you have the right to submit the work under the project's Apache 2.0 license.

The DCO ensures that:
- You have the legal right to submit your contributions
- Your contributions can be distributed under the Apache 2.0 license
- The project maintains clear legal provenance of all code

## How to Contribute

### 1. Use DCO for All Commits
For every commit:
```bash
git commit -s -m "Your commit message"
```
This adds the DCO sign-off to your commit.

### 2. Submit Your Contribution
1. Fork the repository
2. Create your feature branch
3. Make your changes
4. Commit with DCO sign-off
5. Submit a Pull Request

## Verification

Your PR will be checked for:
1. DCO sign-off (every commit)
2. Code quality
3. Tests
4. Security implications
